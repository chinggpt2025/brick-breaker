# `/failure`｜失敗推演

**用途**：反向推演失敗路徑

---

## 執行步驟

### 1. 假設失敗
假設方案已經失敗了

### 2. 倒推原因
列出最可能的 3 個失敗原因：

1. **[原因 1]**
2. **[原因 2]**
3. **[原因 3]**

### 3. 提前規避
指出哪些現在就能提前規避

---

## 解決的盲區

| 盲區類型 | 說明 |
|----------|------|
| 樂觀偏誤 | 總覺得會成功 |
| 黑天鵝忽視 | 不考慮小概率大影響事件 |
| 事後諸葛 | 失敗後才說「早該想到」 |

---

## 輸出格式

```
💀 失敗推演

🎭 假設：方案已經失敗

📉 最可能的 3 個失敗原因：
1. [原因]
2. [原因]
3. [原因]

🛡️ 現在可提前規避的：
- [item]
```
